# Tools

Guidance on how to use provided helper scripts and tools like `such` and `sendtx`. These CLI tools are built when Libdogecoin is configured with `--enable-tools` (default).

See the [libdogecoin repository](https://github.com/dogecoinfoundation/libdogecoin) and `/contrib/` for tool documentation and usage.
