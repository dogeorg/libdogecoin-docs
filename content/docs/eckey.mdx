# Libdogecoin Elliptic Curve Key API

This document explains the basic elliptic curve key API within libdogecoin. ECDSA is used by Dogecoin to ensure that funds can only be spent by their rightful owners. Key concepts include private key, public key, and signature.

See the [libdogecoin repository](https://github.com/dogecoinfoundation/libdogecoin) for the full EC key API reference.
