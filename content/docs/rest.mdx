# Libdogecoin SPV REST API Documentation

This document describes the REST API endpoints exposed by the Libdogecoin SPV node. The API provides access to wallet information (balance, addresses, transactions, UTXOs), as well as wallet and blockchain data.

See the [libdogecoin repository](https://github.com/dogecoinfoundation/libdogecoin) for full endpoint documentation (e.g. GET /getBalance, GET /getAddresses, GET /getTransactions, GET /getUTXOs, and more).
