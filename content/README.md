# Content structure

## Fumadocs (Next.js) — `content/docs/`

- **`.mdx`** files are the source for the Fumadocs site (sidebar, search, TOC).
- All doc pages live here: `what.mdx`, `getting-started.mdx`, `address.mdx`, etc.
- **`meta.json`** defines the sidebar order and section titles.

## Adding or editing content

- Edit or add files in **`content/docs/*.mdx`** and update **`content/docs/meta.json`** if you add new pages or change the sidebar.
